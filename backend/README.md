# How to use:
1. Create and activate a virtual environment:

- Linux:

  ```bash
  python3 -m venv venv && source venv/bin/activate
  ```

- Windows

  ```properties
  python -m venv .venv & .venv/Scripts/activate
  ```

2. Install the requirements:

    ```properties
    pip install -r requirements.txt
    ```

3. Run the app:

    use `start_up.sh`
